run()

Executes necessary functions to check collisions and update the game state. This method is called at regular intervals to ensure the game correctly responds to collisions and other events. The following functions are called: - `checkEnemyCollision()`: Checks for collisions with enemies. - `checkThrowObject()`: Checks for thrown objects. - `checkBottleCollision()`: Checks for collisions with bottles. - `checkCoinCollision()`: Checks for collisions with coins. - `checkCollisionThrow()`: Checks for collisions of thrown objects. - `checkCollisionAbove()`: Checks for collisions with objects above the player. - `updateThrow()`: Updates the status of thrown objects. - `checkEndbossActivation()`: Checks if the endboss should be activated. This method is executed every 100 milliseconds.
