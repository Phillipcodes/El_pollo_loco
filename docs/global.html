setWorldStart()

Initializes and starts a new game world. This function performs the following actions: - Stops any ongoing game intervals. - Removes old key event listeners. - Clears the canvas content. - Destroys the current world state. - Initializes a new game level. - Resets the keyboard state. - Creates a new instance of the `World` class with the current `canvas` and `keyboard`. - Sets up new key event listeners. - Hides the start screen by adding the 'd-none' class. - Removes the border styling from the fullscreen button.
